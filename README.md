 ## 二、项目概述
 本项目为第二十一届全国大学生智能汽车竞赛智慧医疗组参赛作品，参赛车队为寻迹透光车队，基于地瓜机器人RDK开发板完成硬件部署与算法开发。车辆完成赛场规定基础功能开发，可稳定完成赛道循线行驶、主动避障、二维码识别并根据码值判定顺时针/逆时针转向行驶，所有功能均为赛场实测落地功能，无额外拓展功能。

 ## 四、已实现核心功能说明
 ### 4.1 赛道黑线循线功能
 1. 摄像头实时采集赛道画面，通过图像灰度化、阈值分割提取赛道黑色引导线。
 2. 算法计算黑线相对车体中心的偏移量，根据偏移差值动态调节左右轮转速差。
 3. 直道匀速行驶，弯道根据曲率自动微调转向角度，完成全赛道稳定循线。
 ### 4.2 赛道主动避障功能
 1. 视觉识别赛道内静态障碍物，完成障碍物轮廓测距与方位判定。
 2. 判定障碍物位于行驶路径内时，车辆小幅减速，横向微调车体完成避让。
 3. 障碍物脱离行驶区域后，车辆修正航向回归原循线轨迹，恢复行驶速度。
 ### 4.3 二维码识别与定向转向功能
 1. 车辆行驶至二维码识别区域后，减速锁定二维码图像，完成解码识别。
 2. 识别顺时针指令码：车辆执行顺时针环绕转向流程。
 3. 识别逆时针指令码：车辆执行逆时针环绕转向流程。
 4. 转向动作完成后，车辆自动回归主赛道黑线，继续循线行驶。
 ## 五、软件架构说明
 1. 图像采集层：调用摄像头接口，持续推送实时画面流。
 2. 图像处理层：黑线特征提取、障碍物轮廓检测、二维码解析解码。
 3.决策控制层：综合图像信息输出转向、调速、避障、定向动作指令。
 4.底层驱动层：接收上层指令，控制电机驱动完成硬件动作执行。
 ## 六、赛场调试问题与解决方案
 1. 强光环境下黑线识别阈值漂移
 解决方案：增设动态阈值自适应算法，根据环境亮度自动调整二值化参数，保证强光、弱光场景黑线识别稳定。
 2.近距离障碍物识别轮廓粘连
 解决方案：优化图像腐蚀膨胀算子，分离障碍物与赛道背景轮廓，提升近距离避障识别精度。
 3.二维码倾斜状态解码失败
 解决方案：增加图像透视矫正预处理，对倾斜二维码做画面校正，提升多角度下解码成功率。
 ## 七、项目运行环境
 - 主控系统：地瓜RDK配套嵌入式系统
 - 依赖库：OpenCV图像库、pyzbar二维码解析库、串口电机驱动库
 - 运行方式：上电后自动启动主程序，车辆自主完成全流程赛事动作，无需人工二次干预。
